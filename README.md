# zhifou

本项目是一个基于Gradle项目构建工具构建的SpringBoot、Spring Cloud、Spring Cloud Alibaba微服务项目。

## 项目结构

```
zhifou
├── customer-center --客户中心
├── kernel -- 微服务系统公共模块
  ├── kernel-common 
```
